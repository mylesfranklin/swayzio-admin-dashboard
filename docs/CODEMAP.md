# Code Map

> Current as of 2026-07-07. This is a path-level map for agents and reviewers. For behavior and
> decisions, see `docs/ARCHITECTURE.md`, `docs/HANDOFF.md`, and `docs/DECISIONS.md`.

## App Entrypoints

- `src/app/layout.tsx` — root layout, Clerk provider, `data-theme="swayzio"`.
- `src/proxy.ts` — Clerk middleware and public-route exclusions.
- `src/app/(dashboard)/layout.tsx` — founder gate plus dashboard shell.
- `src/app/(auth)/sign-in/[[...sign-in]]/page.tsx` — Clerk sign-in.
- `src/app/(auth)/sign-up/[[...sign-up]]/page.tsx` — Clerk sign-up.
- `src/app/not-authorized/page.tsx` — non-founder access page.

## Implemented Pages

- `src/app/(dashboard)/page.tsx` — main dashboard.
- `src/app/(dashboard)/agent/page.tsx` — Eve chat.
- `src/app/(dashboard)/analytics/stripe/page.tsx` — Stripe analytics.
- `src/app/(dashboard)/analytics/hubspot/page.tsx` — HubSpot analytics.
- `src/app/(dashboard)/database/page.tsx` — Swayzio-Core app DB analytics.
- `src/app/(dashboard)/sync-status/page.tsx` — OS sync freshness and quality checks.
- `src/app/(dashboard)/design-system/page.tsx` — design system browser.

## Implemented API Routes

- `src/app/api/stripe/metrics/route.ts`
- `src/app/api/hubspot/metrics/route.ts`
- `src/app/api/cron/refresh/route.ts`

## Layout And UI

- `src/components/layout/` — shell, sidebar, header, mobile nav, nav config.
- `src/components/ui/` — `Button`, `Badge`, `Card`, copy button, logo, info hint.
- `src/components/charts/` — ApexCharts wrapper and chart primitives.
- `src/components/dashboard/` — main dashboard cards/charts.
- `src/components/stripe/` — Stripe analytics client surface.
- `src/components/hubspot/` — HubSpot analytics client surface.
- `src/components/database/` — database analytics client surface.
- `src/components/sync-status/` — OS sync health and data-quality surface.
- `src/components/agent/agent-chat.tsx` — Eve chat UI.
- `src/components/design-system/` — token/component gallery UI.

## Dashboard Server Modules

- `src/server/cache.ts` — dashboard SWR cache.
- `src/server/db/index.ts` — dashboard Neon client for `DATABASE_URL`.
- `src/server/db/app.ts` — read-only Swayzio-Core app DB client.
- `src/server/integrations/stripe.ts` — Stripe source metrics.
- `src/server/integrations/stripe-dashboard.ts` — cached Stripe aggregate for UI/API.
- `src/server/integrations/hubspot.ts` — HubSpot source metrics.
- `src/server/integrations/hubspot-dashboard.ts` — cached HubSpot aggregate for UI/API.
- `src/server/integrations/app-tracks.ts` — app DB upload trend.
- `src/server/integrations/app-db-stats.ts` — app DB introspection.
- `src/server/os/dashboard.ts` — OS-backed dashboard readers and sync-status queries.

## Auth Helpers

- `src/lib/auth.ts` — Clerk config flags and `isFounder`.
- `src/lib/require-founder.ts` — API route founder guard.

## Swayzio OS

- `src/server/os/db.ts` — owner/admin OS Neon client.
- `src/server/os/sync.ts` — `withSyncRun` run ledger wrapper.
- `src/server/os/load.ts` — raw landing and identity resolution helpers.
- `src/server/os/embed.ts` — embedding helper for memory rows.
- `src/server/os/feeds/stripe.ts` — Stripe customers, subscriptions, catalog, invoices, charges, refunds, balance transactions, and metrics feed.
- `src/server/os/feeds/hubspot.ts` — HubSpot contacts and companies feed; deals intentionally excluded.
- `src/server/os/feeds/app.ts` — Swayzio-Core app raw/core/metrics feed.
- `src/server/os/feeds/mercury.ts` — Mercury accounts, transactions, recipients, categories, cards, statements, organization, users, events, webhooks, and credit/treasury feed.
- `src/server/os/feeds/facebook.ts` — Facebook/Meta Pages, posts, organic insights, ad accounts, campaigns, and Ads Insights feed.
- `db/swayzio-os/migrations/` — migrations `0001` through `0019`.

## Eve Agent

- `agent/agent.ts` — model configuration.
- `agent/channels/eve.ts` — Clerk bearer/founder auth for `/eve/v1/*`.
- `agent/instructions.md` — always-on agent instructions.
- `agent/lib/os.ts` — read-only OS client and embedding helper for tools.
- `agent/lib/format.ts` — JSON-safe coercion and display helpers.
- `agent/tools/*.ts` — read tools plus approval-gated `trigger_sync`.
- `agent/skills/data-dictionary/SKILL.md` — metric/view definitions for the agent.

## Scripts

- `scripts/build-theme.ts` — DESIGN.md to generated CSS/JSON/public assets.
- `scripts/os-migrate.ts` — Swayzio OS migration runner.
- `scripts/os-sync.ts` — Swayzio OS feed orchestrator.
- `scripts/os-verify.ts` — live source vs OS verification.
- `scripts/os-quality.ts` — OS freshness/data-quality gate.
- `scripts/os-embed.ts` — memory doc ingest and embedding backfill.
- `scripts/refresh-stripe.ts`, `scripts/verify-stripe.ts` — Stripe cache probes.
- `scripts/refresh-hubspot.ts`, `scripts/verify-hubspot.ts` — HubSpot cache probes.

## CI And Deployment

- `.github/workflows/ci.yml` — `npm ci`, design lint/check/diff, TypeScript.
- `.github/workflows/os-sync.yml` — six-hour OS feed sync and manual dispatch.
- `vercel.json` — six-hour dashboard cache warm cron.
- `next.config.ts` — Turbopack root, server externals, `withEve`.
